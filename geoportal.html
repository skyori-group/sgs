Гепортал

на платформе SkyoriGIS

Организовать удобный доступ к пространственным данными для разных категорий пользователей целесообразно с помощью геопортала. Геопортал - это единая точка доступа к геоданным, инструмент визуализации и анализа данных. Платформа SkyoriGIS (Скайори ГИС) - отечественная разработка, предоставляет возможности для создания геоинформационных порталов, полнофункциональных Веб ГИС.

Возможности Геопортала для работы с геоданными

  • единая точка доступа к геоданным
  • пространственный анализ и мониторинг объектов
  • отчеты по выбранным участкам карты
  • интерактивная веб-визуализация
  • организация удобного каталога геоданных
  • поиск и фильтрация
  • интеграция с настольными и серверными ГИС-продуктами
  • обмен данными с информационными системами предприятия
  • настройка прав доступа для разных категорий пользователей

Функционал Геопорталов на платформе SkyoriGIS

  • Управление картами: создание/изменение/удаление карт
  • Создание новых слоев (генерация таблицы пространственных данных в БГД, генерация слоя в ГИС-сервере)
  • Добавление слоев из внешних источников
  • Загрузка данных из файлов с возможностью проверки
  • Отображение перечня слоев и карт в виде каталога с возможностью фильтрации и поиска

На слоях карты можно настраивать:

  • внешний вид слоя для разных значений зумов: стиль отображения слоя, стиль подписей слоя
  • возможности идентификации объектов слоя на карте
  • поиск, отображение атрибутов
  • порядок слоев в дереве, слоев на карте

Возможности Веб-карты

  • Навигация по карте
  • Управление содержанием карты
  • Поиск по карте
  • Измерения на карте: определение координат выбранной точки, измерение расстояний и площадей
  • Экспорт данных карты в файлы Esri Shape, MapInfo Tab, Mid/Mif, KML, Geojson, GPX, CSV
  • Редактирование объектов карты
  • Инструменты анализа на карте: поиск пересечений объектов с выбранными слоями, сравнение объектов, слоев и др.
